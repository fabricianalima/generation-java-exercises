Este repositório contém exercícios práticos desenvolvidos durante o bootcamp de Desenvolvimento Java da Generation Brasil e será atualizado conforme avanço no bootcamp, incluindo novos conceitos, desafios e projetos práticos.

---
## 🧠 Conteúdos Abordados

### 🔹 Fundamentos de Programação
- Variáveis, tipos de dados e operadores
- Entrada e saída de dados (Scanner)
- Estruturas condicionais (if, switch)
- Laços de repetição (for, while, do-while)

### 🔹 Estruturas de Dados
- Vetores e matrizes
- Collections
- Manipulação de dados

### 🔹 Programação Orientada a Objetos (POO)
- Classes e objetos
- Encapsulamento
- Herança e polimorfismo
- Métodos

### 🔹 Tratamento de Erros
- Exceptions
- Boas práticas de validação

### 🔹 Java Avançado
- Streams
- Optional

### 🔹 Banco de Dados
- Modelagem de dados
- SQL (consultas e manipulação)

### 🔹 Desenvolvimento Backend
- Spring Boot
- APIs REST
- Spring Data JPA

### 🔹 Testes
- Testes unitários
- Qualidade de software

---

## 💻 Tecnologias Utilizadas

- Java
- Spring Boot
- MySQL
- Git e GitHub
- STS (Spring Tool Suite)

---
## 🎯 Objetivo

Desenvolver uma base sólida em Java e evoluir continuamente para atuar como desenvolvedora backend, aplicando boas práticas de programação e construção de APIs.
